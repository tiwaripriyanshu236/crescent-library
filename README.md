# 📚 Crescent Library

A modern and responsive Library Booking Website built with **TanStack Start**, **React**, **TypeScript**, **Tailwind CSS**, **Supabase**, and **Resend**.

## 🚀 Live Demo

🌐 https://crescent-library.vercel.app/

## 📂 GitHub Repository

🔗 https://github.com/tiwaripriyanshu236/crescent-library

## ✨ Features

- 📚 Modern Library Website
- 💺 Online Seat Booking
- 📅 Daily / Weekly / Monthly Plans
- 📧 Admin Email Notifications
- 🗄️ Supabase Database
- 📱 Responsive Design
- 🌙 Dark Theme
- 📞 Contact Form
- 🖼️ Gallery Section
- ❓ FAQ Section
- ⭐ Testimonials
- ⚡ Fast Performance

## 🛠️ Tech Stack

- React
- TypeScript
- TanStack Start
- Tailwind CSS
- Shadcn UI
- Supabase (PostgreSQL)
- Resend API
- Vercel

## ⚙️ Installation

```bash
git clone https://github.com/tiwaripriyanshu236/crescent-library.git
cd crescent-library
bun install
bun run dev
```
